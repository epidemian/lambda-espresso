## Development

Install Node.js and NPM.

Install CoffeeScript and Jison:

```bash
sudo npm install -g coffee-script jison
```

Compile and watch the .coffee files:

```bash
coffee -wc .
```

Compile the grammar file:

```bash
jison grammar.jison
```
